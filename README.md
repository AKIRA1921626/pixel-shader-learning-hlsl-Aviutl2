# pixel-shader-learning-hlsl-Aviutl2
HLSLのピクセルシェーダーを学ぶためのAviutl2用のスクリプトです。  
私自身が学習を行うために、AIに作ってもらいました。  
UI辺りは自分である程度指示をしました。  
  
**使い方  
このファイルをメモ帳などで開き、HLSL内の「//」を付けたり消したりして上書き保存し、  
AviUtl上の画面構成がどう変わるかを観察してみてください。**  
  
スクリプトの中身を見てファイルを書き換えならが学習する用途ですので、ある程度のluaでのスクリプト作成経験があることを前提としています。  
また、動画編集自体には一切役に立ちません。  
赤、緑、青、透明度のトラックバーの数値は0~100となっていますが、スクリプト内では/100しているため、実際には0~1の範囲で処理を行う用になっています。  
例:52の場合は52/100で、0.52として扱われています。  

トラックバーの操作画面などはこちらである程度指定しましたが、内容の説明などはAIに書いてもらった部分が大半ですので、裏は取れていません。  

**二次配布可、改変可、クレジット不要**
